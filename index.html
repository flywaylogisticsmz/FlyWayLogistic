Link para partilhar com os clientes
⚠️ Importante: Envie este link aos seus clientes (WhatsApp, email, etc). Quando abrirem este link, conseguirão rastrear as suas encomendas automaticamente, porque o endereço já tem o código do Google Sheets incorporado.
1
Criar o Google Sheet
Aceda a sheets.google.com e crie uma folha nova. O nome não importa — o script cria a aba Encomendas automaticamente.
2
Abrir Apps Script e colar o código
No Google Sheet vá a Extensões → Apps Script. Apague tudo e cole o código do ficheiro appsscript.gs. Clique em Guardar (Ctrl+S).
3
Publicar como Web App
No editor Apps Script clique em Implementar → Nova implementação. Selecione tipo "Aplicação Web". Configure:
▸ Executar como: Eu mesmo
▸ Quem tem acesso: Qualquer pessoa ← obrigatório!
▸ Clique Implementar → autorize → copie o URL gerado
4
Colar o URL aqui e ativar
Cole o URL do Apps Script abaixo, teste a ligação e clique em Guardar e Ativar. Depois copie o Link de Partilha e envie aos clientes.
